# claudecode

A monorepo for experimental projects and ideas.

## Structure

```
claudecode/
├── projects/
│   └── task-tracker/   # Next.js task tracker app
│   └── ...             # Future projects
```

## Projects

### Task Tracker
A simple task tracking app built with Next.js, TypeScript, and Tailwind CSS.

```bash
cd projects/task-tracker
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Claude Code GitHub Action

This repo has the Claude Code GitHub Action installed. Mention `@claude` in issues or PRs to get AI assistance.
